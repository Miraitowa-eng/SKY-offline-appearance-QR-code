# 光遇扫码身高解析插件

发送 `sky.thatg.co/o=` 开头的扫码链接即可解析。

注意：这个项目不是独立机器人程序，而是 AstrBot 插件。需要先有一个已经运行的 AstrBot，并且这个插件已经安装到 AstrBot 里。用户在 AstrBot 已接入的平台聊天窗口里发送链接后，AstrBot 会自动回复解析结果。

## 玩家获取链接步骤

1. 打开《光遇》。
2. 点击右上角齿轮，进入设置。
3. 点击 `更多`。
4. 往下拉，找到 `线下活动外观二维码`。
5. 用手机扫码。
6. 手机会得到一个 `https://sky.thatg.co/o=...` 开头的链接。
7. 把这个链接发送到已经安装本插件的 AstrBot 聊天窗口。
8. AstrBot 会返回身高参数信息和装扮 ID。

示例：

```text
https://sky.thatg.co/o=8RZ7...
```

返回字段：

```text
体型值
身高值
当前身高
最高身高
最矮身高
查询时间
装扮信息
```

解码流程为：`Base64 -> LZ4 block -> JSON`，完全离线，不请求第三方接口，也不会消费好友码。

默认计算公式：

```text
当前身高 = 7.6 - 8.3 * scale - 3 * height
最高身高 = 7.6 - 8.3 * scale - 3 * 2
最矮身高 = 7.6 - 8.3 * scale - 3 * -2
```

这里的“身高”是常见的光遇身高编号口径，数值越小越高。

公式参考：<https://wiki.biligame.com/sky/%E8%BA%AB%E9%AB%98>
