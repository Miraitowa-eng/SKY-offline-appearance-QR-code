# 光遇扫码身高解析插件

发送 `sky.thatg.co/o=` 开头的扫码链接即可解析。

示例：

```text
https://sky.thatg.co/o=8RZ7...
```

返回字段：

```text
体型值
身高值
当前身高编号
最高身高编号
最矮身高编号
查询时间
装扮信息
```

解码流程为：`Base64 -> LZ4 block -> JSON`，完全离线，不请求第三方接口，也不会消费好友码。

默认计算公式：

```text
当前身高编号 = 7.6 - 8.3 * scale - 3 * height
最高身高编号 = 7.6 - 8.3 * scale - 3 * 2
最矮身高编号 = 7.6 - 8.3 * scale - 3 * -2
```

这里的“身高编号”是常见的光遇身高编号口径，数值越小越高。

公式参考：<https://wiki.biligame.com/sky/%E8%BA%AB%E9%AB%98>

完整安装说明请看 `README.md`。
